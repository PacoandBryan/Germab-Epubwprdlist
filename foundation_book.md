# Session 1

*   **The Word:** unerlässlich
*   **Translation:** essential / indispensable
*   **B2 Example Sentence:** Eine gute Vorbereitung ist für den Erfolg dieses Projekts unerlässlich.
*   **Sentence Translation:** Good preparation is essential for the success of this project.

*   **The Word:** bewerten (hat bewertet)
*   **Translation:** to evaluate / to assess
*   **Verb Forms:** Präteritum: bewertete, Perfekt: hat bewertet
*   **B2 Example Sentence:** Wir müssen die aktuellen Marktdaten sorgfältig bewerten, bevor wir eine Entscheidung treffen.
*   **Sentence Translation:** We must carefully evaluate the current market data before making a decision.
